What is C++?

C++ is a middle-level programming language developed by Bjarne Stroustrup starting in 1979 at Bell Labs. C++ runs on a variety of platforms, such as Windows, Mac OS, and the various versions of UNIX. C++ is an extension of the C programming language with object-oriented programming concepts. Or, we can say, "C++ is a superset of C programming with additional implementation of object-oriented concepts".

Why Learn C++?

C++ is a powerful programming language that is essential for students and professionals who want to become skilled software developers. Here are some key reasons why learning C++ is valuable:

  • Close to Hardware: C++ allows direct interaction with hardware, providing better control over memory management and performance optimization.
  • Strong Object-Oriented Programming (OOP) Concepts: Helps in understanding OOP fundamentals like polymorphism, inheritance, and encapsulation.
  • High Demand & Job Security: C++ is a widely used language, and skilled developers are highly paid.
  • Used in System & Application Development: Essential for game development, operating systems, and high-performance applications.
  • Deep Understanding of Core Programming Concepts: Covers compilers, linkers, loaders, data types, and storage classes.

C++ Overview

C++ is a statically typed, compiled, general-purpose, case-sensitive, free-form programming language that supports procedural, object-oriented, and generic programming.

Key Features of C++

  • Object-Oriented: C++ supports OOP concepts like classes, objects, inheritance, polymorphism, and encapsulation.
  • Portable: C++ programs can run on different platforms with little or no modification.
  • Mid-level Language: Combines features of both high-level and low-level languages.
  • Structured Programming Language: Supports breaking down programs into functions and modules.
  • Rich Library Support: Comes with a rich set of libraries (Standard Template Library).
  • Memory Management: Provides direct control over memory through pointers.

C++ vs C Comparison

Feature C C++
Programming Paradigm Procedural Multi-paradigm
OOP Support No Yes
Standard Library Limited Extensive (STL)

Your First C++ Program

Let's write and understand a simple "Hello, World!" program in C++.

#include <iostream>
using namespace std;

// main() is where program execution begins
int main() {
    cout << "Hello, World!"; // prints Hello, World!
    return 0;
}

Code Explanation

  • #include <iostream>: This is a header file library that lets us work with input and output objects.
  • using namespace std: Means we can use names for objects and variables from the standard library.
  • int main(): This is called a function. Any code inside its curly brackets {} will be executed.
  • cout: (pronounced "see-out") is an object used to output/print text.
  • return 0: Ends the main() function and returns the value 0 to the calling process.

Quick Quiz

What is the purpose of the #include directive?
To include header files
To define variables
To create functions
Correct! The #include directive is used to include header files that contain declarations of functions and objects.

C++ Numeric Data Types

C++ provides various numeric data types to represent numbers in different forms and sizes.

Fundamental Numeric Types

Data Type Size (bytes) Description
int 4 Stores whole numbers without decimals
float 4 Stores fractional numbers (6-7 decimal digits)
double 8 Stores fractional numbers (15 decimal digits)
short 2 Small integer (-32,768 to 32,767)
long 4 or 8 Larger version of int

Example Code

#include <iostream>
using namespace std;

int main() {
    // Integer types
    int myNum = 1000;
    short myShort = 500;
    long myLong = 150000;
    
    // Floating point types
    float myFloat = 5.75;
    double myDouble = 19.99;
    
    cout << "int: " << myNum << endl;
    cout << "short: " << myShort << endl;
    cout << "long: " << myLong << endl;
    cout << "float: " << myFloat << endl;
    cout << "double: " << myDouble << endl;
    
    return 0;
}
Output:
int: 1000
short: 500
long: 150000
float: 5.75
double: 19.99

Quick Quiz

Which data type would you use to store a number like 3.141592653589793?
float
double
int
Correct! The double type has about 15 decimal digits of precision, making it suitable for this value.

C++ if Statement

The if statement is used to specify a block of code to be executed if a condition is true.

Syntax

if (condition) {
    // code to be executed if condition is true
}

Example

#include <iostream>
using namespace std;

int main() {
    int time = 20;
    
    if (time < 18) {
        cout << "Good day.";
    }
    
    cout << "Program continues...";
    return 0;
}
Output: Program continues...

C++ for Loop

The for loop is used when you know exactly how many times you want to loop through a block of code.

Syntax

for (initialization; condition; increment) {
    // code block to be executed
}

Example

#include <iostream>
using namespace std;

int main() {
    for (int i = 0; i < 5; i++) {
        cout << i << endl;
    }
    return 0;
}
Output:
0
1
2
3
4

Nested for Loop

You can place a loop inside another loop to create nested loops.

#include <iostream>
using namespace std;

int main() {
    for (int i = 1; i <= 3; i++) {
        for (int j = 1; j <= 3; j++) {
            cout << "i=" << i << ", j=" << j << endl;
        }
    }
    return 0;
}

Quick Quiz

How many times will the inner loop execute in the nested example above?
3
9
6
Correct! The inner loop runs 3 times for each of the 3 iterations of the outer loop, totaling 9 executions.

Object-Oriented Programming in C++

OOP is a programming paradigm that uses objects and classes to organize code. C++ supports these OOP concepts:

1. Classes and Objects

A class is a blueprint for creating objects (a particular data structure).

class MyClass {       // The class
  public:             // Access specifier
    int myNum;        // Attribute (int variable)
    string myString;  // Attribute (string variable)
};

int main() {
    MyClass myObj;  // Create an object of MyClass
    
    // Access attributes and set values
    myObj.myNum = 15;
    myObj.myString = "Some text";
    
    cout << myObj.myNum << endl;
    cout << myObj.myString << endl;
    return 0;
}

2. Inheritance

Inheritance allows us to define a class in terms of another class.

// Base class
class Vehicle {
  public:
    string brand = "Ford";
    void honk() {
        cout << "Tuut, tuut!" << endl;
    }
};

// Derived class
class Car: public Vehicle {
  public:
    string model = "Mustang";
};

int main() {
    Car myCar;
    myCar.honk();
    cout << myCar.brand + " " + myCar.model << endl;
    return 0;
}

3. Polymorphism

Polymorphism allows methods to do different things based on the object that is acting upon them.

class Animal {
  public:
    void animalSound() {
        cout << "The animal makes a sound" << endl;
    }
};

class Pig : public Animal {
  public:
    void animalSound() {
        cout << "The pig says: wee wee" << endl;
    }
};

class Dog : public Animal {
  public:
    void animalSound() {
        cout << "The dog says: bow wow" << endl;
    }
};

int main() {
    Animal myAnimal;
    Pig myPig;
    Dog myDog;
    
    myAnimal.animalSound();
    myPig.animalSound();
    myDog.animalSound();
    return 0;
}

Quick Quiz

Which OOP concept allows a class to inherit properties from another class?
Inheritance
Polymorphism
Encapsulation
Correct! Inheritance allows a class to inherit properties and methods from another class.

C++ Pointers

Pointers are variables that store memory addresses...

#include <iostream>
using namespace std;

int main() {
    int var = 5;
    int* ptr = &var;
    
    cout << "Value: " << *ptr << endl;
    cout << "Address: " << ptr << endl;
    return 0;
}
